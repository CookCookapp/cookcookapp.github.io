# CookCook - 스마트 요리 추천 웹앱

당신의 재료로 요리를 추천하고, 단계별 알림으로 완벽한 요리를 만들어보세요!

## 주요 기능

### 1. 식재료 관리 대시보드
- 현재 보유 중인 식재료와 양념을 체계적으로 관리
- 카테고리별 분류 (채소, 고기, 해산물, 양념, 곡물, 유제품, 기타)
- 재료 추가, 수정, 삭제 기능
- 로컬 스토리지를 활용한 데이터 저장

### 2. 스마트 요리 추천
- 보유 재료 기반 자동 매칭 시스템
- 재료 일치율을 백분율로 표시
- 난이도, 조리 시간, 인분 정보 제공
- 다양한 한식, 중식, 양식, 동남아식 레시피

### 3. 조리 타이머 & 알림 시스템
- 단계별 조리 가이드
- 실시간 타이머 기능
- 중요 단계에서 브라우저 알림 자동 발송
- 조리 진행 상황 시각화
- 일시정지/재개 기능
- 이전/다음 단계 이동

### 4. 요리 검색
- 이름, 재료, 설명으로 검색
- 요리 종류 및 난이도 필터링
- 실시간 검색 결과 표시

### 5. AI 요리 추천
- 자연어로 요리 질문 가능
- 예: "싱가포르식 돼지고기 조리법을 알려줘"
- 지능형 키워드 매칭 시스템
- 맥락 기반 레시피 추천

### 6. 광고 시스템 (비즈니스 모델)
- 배너 광고 영역
- 제품 추천 사이드바
- 구매 링크를 통한 수수료 수익 모델

## 기술 스택

- **Frontend**: React 18 + TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **State Management**: React Context API
- **Storage**: LocalStorage

## 설치 및 실행

### 필수 요구사항
- Node.js 16 이상
- npm 또는 yarn

### 설치 방법

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```

개발 서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.

## 사용 방법

### 1. 재료 등록하기
1. 대시보드 탭으로 이동
2. "재료 추가" 버튼 클릭
3. 재료명, 수량, 단위, 카테고리 입력
4. 추가 버튼 클릭

### 2. 요리 추천 받기
1. "요리 추천" 탭으로 이동
2. 보유 재료와 일치하는 레시피 확인
3. 일치율이 높은 순으로 정렬됨
4. "조리 시작하기" 버튼 클릭

### 3. 조리하기
1. 레시피 선택 후 조리 시작
2. 각 단계의 지시사항 따라하기
3. 알림이 설정된 단계에서는 브라우저 알림 수신
4. 다음/이전 버튼으로 단계 이동
5. 완료 버튼으로 조리 종료

### 4. AI 추천 사용하기
1. "AI 추천" 탭으로 이동
2. 원하는 요리를 자연어로 입력
   - 예: "싱가포르식 돼지고기 조리법을 알려줘"
3. AI가 관련 레시피 추천
4. 마음에 드는 레시피 선택

## 브라우저 알림 설정

조리 타이머의 알림 기능을 사용하려면:
1. 브라우저에서 알림 권한 허용
2. 조리 시작 시 자동으로 권한 요청
3. 중요 단계에서 알림 자동 발송

## 샘플 레시피

현재 포함된 레시피:
- 김치볶음밥 (한식, 쉬움, 15분)
- 간장치킨 (한식, 보통, 40분)
- 싱가포르식 돼지고기 볶음 (싱가포르, 보통, 35분)
- 토마토 파스타 (이탈리안, 쉬움, 25분)
- 계란볶음밥 (중식, 쉬움, 10분)
